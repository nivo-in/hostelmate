# HostelMate — by Nivo

Smart hostel management platform. Replaces manual registers with a
digital system for attendance, leaves, complaints, mess, and parent tracking.

## Apps
- `apps/client` — React + Vite frontend
- `apps/server` — Node.js + Express backend

## Packages
- `packages/db` — Mongoose schemas
- `packages/auth` — Firebase auth helpers
- `packages/ui` — Shared UI components

## Getting started
cp .env.example .env
pnpm install
pnpm dev
